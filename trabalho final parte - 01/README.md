# Documentação do Projeto – Programação Web I

Este arquivo apresenta as orientações necessárias para executar o projeto desenvolvido para a disciplina de Programação Web I, bem como a forma correta de acessá-lo no ambiente local.

# 1. Como executar o projeto

O sistema foi construído para rodar exclusivamente em um ambiente local e depende de um servidor Apache ativo para funcionar. Não há hospedagem online; portanto, todo o acesso deve ser feito diretamente pela máquina onde o projeto está instalado.

# 1.1 Requisitos indispensáveis

- Antes de iniciar o acesso, verifique se os seguintes itens estão instalados:
- Servidor Apache (como parte do XAMPP)
- Navegador web instalado
- Projeto colocado na pasta utilizada pelo Apache para servir arquivos (“htdocs”)

# 1.2 Procedimento de acesso

- Após confirmar os requisitos, siga os passos abaixo:
- Ative o módulo Apache no painel da ferramenta utilizada (XAMPP, WAMP etc.).
- Abra seu navegador.
- Digite ou cole o endereço abaixo na barra de pesquisa:

http://localhost/Desenvolvimento-WEB--1/Desenvolvimento-WEB-/Trabalho%20Final/Public/HTML/index.html


# 2. Sobre o sistema desenvolvido

O projeto consiste em um conjunto de páginas e scripts que compõem a aplicação proposta como trabalho final. Ele utiliza HTML, CSS e JavaScript para a camada visual e conta com PHP e banco de dados para tratamento das funcionalidades internas.

 Entre as operações presentes no sistema estão:

- Exibição e processamento de conteúdo
- Integração com backend
- Consultas e manipulação de dados


# 3. Tecnologias empregadas

A aplicação foi construída com:

- HTML5
- CSS3
- JavaScript
- PHP
- Banco de dados (PostgreSQL)
- Servidor Apache
